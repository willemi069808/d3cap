# d3cap
Showcase using D3js in SAP CAP NodeJs

Using d3js vizualisations (https://d3js.org/) inside a SAP CAP application
Contains 2 sample charts with demo data: Bollinger and Sankey

Usage:
- git clone
- cd d3cap
- npm install
- cds watch
